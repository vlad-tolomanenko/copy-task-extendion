# 🚀 Quick Installation Guide

## For Windows & macOS

### Step 1: Extract the ZIP
1. Download `asana-copy-extension.zip`
2. Extract it to any folder on your computer

### Step 2: Install in Chrome
1. Open Chrome and go to: `chrome://extensions/`
2. Turn ON "Developer mode" (toggle in top-right corner)
3. Click "Load unpacked"
4. Select the extracted `asana-copy-extension` folder
5. Done! ✓

### Step 3: Use it!
1. Go to any Asana task
2. Press `Alt+Shift+C`
3. Paste anywhere - the link is embedded!

## First Time Setup

1. Click the extension icon (📋)
2. Go to Settings (gear icon)
3. Choose your language
4. Pick your preferred format (Markdown recommended)
5. All other features work out of the box!

## Troubleshooting

**Not working?**
1. Open Settings
2. Enable "Enable logging"
3. Try copying again
4. Click "View Logs" to see what's wrong

**Can't find task name?**
- Make sure you're on a specific Asana task page (not the task list)
- The URL should look like: `app.asana.com/0/home/...../TASK_ID`
- Reload the page
